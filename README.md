## Run locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```
Go to the app or api folder

```bash
  cd app
```
```bash
  cd api
```

Install the dependencies

```bash
  npm install
```

Run the server

```bash
  npm start
```
